# Jogo de Trivia (Python)

Projeto desenvolvido utilizando:
- API OpenTDB (https://opentdb.com/)
- API de tradução de exemplo da professora
- Python + requests

## Funcionalidades

- Seleção de categoria
- Seleção de dificuldade
- 7 questões por partida
- Tradução da pergunta e das respostas
- Verificação de acertos e erros
- Placar final
